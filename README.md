# 🤖 MyProject - AI Platform

एक शक्तिशाली AI platform जिसमें Chat Bot, Image Generation, और Face Swap Video Features हैं।

## ✨ Features

### 1. 💬 Chat Bot
- किसी भी सवाल का जवाब दें
- हिंदी और अंग्रेजी दोनों में काम करे
- Hugging Face का उपयोग करता है

### 2. 🖼️ Image Generator
- Text से Image बनाएं
- Stable Diffusion का उपयोग
- कोई भी description दे सकते हैं

### 3. 🎥 Face Swap / Video Reenactment
- एक video का reference लें
- दूसरे video में face swap करें
- Advanced AI technology का उपयोग
